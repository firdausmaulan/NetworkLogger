1. Add jitpack.io to your project
```
  allprojects {
     repositories {
        maven { url 'https://jitpack.io' }
     }
  }
```

2. Add simple barcode scanner to your project
```
  dependencies {
     implementation 'com.github.firdausmaulan:NetworkLogger:1.7.3'
  }
```
